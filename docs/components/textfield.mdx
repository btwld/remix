A text input component for collecting user text input with labels, hints, and helper text.

## When to use this

- **Form inputs**: Collect text data in forms (names, emails, addresses)
- **Search fields**: Enable users to enter search queries
- **Comments**: Allow users to enter text comments or feedback
- **Data entry**: Capture any text-based user input

## Basic implementation

<CodeGroup title="Basic implementation" defaultLanguage="dart">
```dart
import 'package:flutter/material.dart';
import 'package:remix/remix.dart';

class TextfieldExample extends StatefulWidget {
  const TextfieldExample({super.key});

  @override
  State<TextfieldExample> createState() => _TextfieldExampleState();
}

class _TextfieldExampleState extends State<TextfieldExample> {
  final controller = TextEditingController();

  @override
  Widget build(BuildContext context) {
    return SizedBox(
      width: 300,
      child: RemixTextField(
        controller: controller,
        hintText: 'Placeholder',
        label: 'Label',
        helperText: 'Required field',
        style: style,
      ),
    );
  }

  RemixTextFieldStyle get style {
    return RemixTextFieldStyle()
        .color(Colors.grey.shade800)
        .backgroundColor(Colors.white)
        .borderRadiusAll(const Radius.circular(8.0))
        .height(44)
        .paddingX(12)
        .spacing(8)
        .label(
          TextStyler()
              .color(Colors.blueGrey.shade900)
              .fontWeight(FontWeight.w500),
        )
        .helperText(
          TextStyler()
              .fontWeight(FontWeight.w300)
              .color(Colors.blueGrey.shade600),
        )
        .hintColor(Colors.blueGrey.shade500)
        .shadow(
          BoxShadowMix()
              .blurRadius(1)
              .color(Colors.black12)
              .offset(const Offset(0, 1)),
        )
        .border(
          BoxBorderMix.all(BorderSideMix(color: Colors.grey.shade300)),
        )
        .onFocused(
          RemixTextFieldStyle().border(
            BoxBorderMix.all(
              BorderSideMix()
                  .color(Colors.deepPurpleAccent)
                  .width(3)
                  .strokeAlign(BorderSide.strokeAlignCenter),
            ),
          ),
        );
  }
}
```
</CodeGroup>

## Properties

### Widget Properties

#### `controller` → `TextEditingController?`

Controls the text being edited.

#### `focusNode` → `FocusNode?`

Optional focus node to control focus behavior.

#### `label` → `String?`

Optional label text displayed above the text field.

#### `hintText` → `String?`

Optional hint text displayed when the field is empty.

#### `helperText` → `String?`

Optional helper text displayed below the text field.

#### `errorText` → `String?`

Optional error text displayed below the text field.

#### `keyboardType` → `TextInputType?`

The type of keyboard to display.

#### `textInputAction` → `TextInputAction?`

The action button on the keyboard.

#### `obscureText` → `bool`

Whether to obscure the text (for passwords). Defaults to `false`.

#### `enabled` → `bool`

Whether the text field is enabled. Defaults to `true`.

#### `readOnly` → `bool`

Whether the text field is read-only. Defaults to `false`.

#### `autofocus` → `bool`

Whether to automatically focus when first built. Defaults to `false`.

#### `maxLines` → `int?`

Maximum number of lines. Defaults to `1`.

#### `maxLength` → `int?`

Maximum number of characters.

#### `onChanged` → `ValueChanged<String>?`

Called when the text changes.

#### `onEditingComplete` → `VoidCallback?`

Called when editing is complete.

#### `onSubmitted` → `ValueChanged<String>?`

Called when the user submits the text.

#### `style` → `RemixTextFieldStyle`

Style configuration. Defaults to `RemixTextFieldStyle.create()`.

#### `styleSpec` → `RemixTextFieldSpec?`

Optional pre-resolved style specification for performance optimization.

### Style Properties

The `RemixTextFieldStyle` class provides a fluent API for styling:

#### Container Styling

**`padding(EdgeInsetsGeometryMix value)`** - Sets padding around text field content.

**`paddingX(double value)`** - Sets horizontal padding.

**`color(Color value)`** - Sets the text color.

**`backgroundColor(Color value)`** - Sets the background color.

**`borderRadiusAll(Radius value)`** - Sets uniform border radius.

**`border(BoxBorderMix value)`** - Sets border styling.

**`height(double value)`** - Sets the text field height.

**`spacing(double value)`** - Sets spacing between elements.

**`shadow(BoxShadowMix value)`** - Adds shadow effects.

#### Text Styling

**`hintColor(Color value)`** - Sets hint text color.

**`labelColor(Color value)`** - Sets label text color.

#### Label & Helper Styling

**`label(TextStyler value)`** - Full control over label text styling.

**`helperText(TextStyler value)`** - Full control over helper text styling.

**`errorText(TextStyler value)`** - Full control over error text styling.

#### Interactive States

**`onFocused(RemixTextFieldStyle style)`** - Applies styles when focused.

**`onDisabled(RemixTextFieldStyle style)`** - Applies styles when disabled.

**`onError(RemixTextFieldStyle style)`** - Applies styles when there's an error.
